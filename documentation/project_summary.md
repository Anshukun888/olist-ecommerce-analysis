# Olist E-commerce Sales Analysis

## Objective
The goal of this project is to analyze the Olist Brazilian e-commerce dataset to understand sales performance, customer behavior, delivery efficiency, and product category trends.

## Dataset
- Source: Kaggle (Olist Brazilian E-commerce Dataset)
- Size: ~100,000+ orders
- Structure: Multiple relational tables (customers, orders, products, payments, sellers, reviews)

## Tools Used
- Microsoft Excel (Data Cleaning & Transformation)
- MySQL (Data Storage & SQL Analysis)
- Power BI (Dashboard Development)
- SQL (Data Extraction & Business Analysis)

## Process Workflow
1. Collected dataset from Kaggle
2. Cleaned data using Excel Power Query
3. Loaded structured data into MySQL
4. Performed SQL-based analysis
5. Built interactive Power BI dashboards

## Key Focus Areas
- Revenue analysis
- Customer behavior
- Delivery performance
- Product category performance
- Payment trends
